# Proyecto PHP

## Descripción
Este proyecto contiene diferentes ejercicios y prácticas desarrolladas en **PHP**.  
El objetivo es aplicar conceptos básicos de programación web como el uso de variables, estructuras de control, operadores y lógica de programación.

## Prerrequisitos
Para ejecutar este proyecto es necesario tener instalado:

- PHP
- Apache (u otro servidor web)
- Git
- Navegador web
